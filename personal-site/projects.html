Projects

I have done several projects focusing on full-stack development, in this case, my projects are:

Countries Web

This project is a simple website that contains data from API called RestCountries API. This website was built in React JS, to create the components, and Native JavaScript to use the fetch API and the filters for the data.

Weather-APP:

This website is a challenge from a devchallenges.io and contains a weather view for the cities in the world. This can get the current location and show the local weather using the weather API. This was built in the React framework, Next JS 13, which implements new features in website routing.

University-app

This is a project that was assigned to me in the FUNVAL course for Final Project, this project consists of a CRUD for a university. This CRUD contains data about the teachers, courses, and students in that university, and I need to create a database, the backend system, and the view for the client. I created this system using Laravel, a PHP framework for this project.
