Skills

I consider myself a self-learner and I love learning about programming. The next skills are several of my current capabilities that I have developed over the years of web development.
Front end skills For the part to develop the website on the client’s side or front end, I use the next tools to organize, stylize and add interactivity to the site.
HTML, CSS, JavaScript This is the basic for frontend, HTML to mark and organize the information, CSS to add styles to the site, and JavaScript to add control and interactivity, to manipulate the client events such as a click or resize the site.
React JS React JS makes me easier to create and recycle components, instead to copy and paste the same code or create templates in HTML. Besides, React JS is reactive, and manages the states of the app and the components, allowing me to add special behaviors when the component appears or dismounts from the browser.
Bootstrap y Tailwind Bootstrap and Tailwind are tools to add styles avoiding writing a lot of CSS, but that had already made the classes and styles for that. To be honest, I like better Tailwind than Bootstrap, because is more flexible to use, in exchange for Bootstrap is so static and restrictive, but a good point is that has components like cards, an images carousel, and so on to import.
Next JS Next JS is the most important React framework. This is so easy to use in the current version 13 with its new app routing. This works much the front end as the back end.
Back end Skills The back end is the server side. This part works with the client’s requests, routing, database connection, user authentication, and other parts of the security of the website.
PHP y MySQL PHP and MySQL work well together, so much so that PHP has special functions for MySQL. MySQL is the System to manage the database while PHP is responsible to manage the requests and the logic for the site
Laravel It is the framework for PHP. This has many tools to develop faster a web site.

Projects

I have done several projects focusing on full-stack development, in this case, my projects are:
Countries Web: This project is a simple website that contains data from API called RestCountries API. This website was built in React JS, to create the components, and Native JavaScript to use the fetch API and the filters for the data.
Weather-APP: This website is a challenge from a devchallenges.io and contains a weather view for the cities in the world. This can get the current location and show the local weather using the weather API. This was built in the React framework, Next JS 13, which implements new features in website routing.
University-app: This is a project that was assigned to me in the FUNVAL course for Final Project, this project consists of a CRUD for a university. This CRUD contains data about the teachers, courses, and students in that university, and I need to create a database, the backend system, and the view for the client. I created this system using Laravel, a PHP framework for this project.
